## [1.0.0] — 2026-06-18

First public release. Packages the Open Collider protocol as an installable Claude skill.

### Added
- `skills/open-collider/SKILL.md` — the canonical skill body (English): the 5-stage protocol (briefing → domain bank → isolated collisions → cross-collision → 3-filter curation → ranked pool), embedded falsifiers, cost gate, anti-patterns, and declared limitations.
- `skills/open-collider/SKILL.pt-BR.md` — Portuguese reference translation.
- `README.md` — landing page: the problem (Artificial Hivemind), the mechanism (bisociation), how it works (Mermaid flow), when to use / not, a worked-example teaser, install, and the science with credits.
- `examples/personal-brand-collision.md` — a full worked run: domain bank of six, an isolated collision, a cross-collision, a ranked pool of 8 angles, discards, and the falsifier checks.
- `templates/brief-template.md` — a fill-in briefing template for Stage 1.
- `open-collider.skill` — packaged skill at the repo root for no-terminal install (Cowork / claude.ai).
- `assets/social-preview.html` — DS V3.0 source for the social card.
