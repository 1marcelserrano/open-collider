# Examples

Worked open-collider runs — read these to see what the skill produces.

- [personal-brand-collision.md](./personal-brand-collision.md) — a full run on
  the brief *"original angles for an essay on building a personal brand without
  sounding like everyone else."* Domain bank of six (mycology, Fresnel optics,
  sourdough, heraldry, free-diving, modal jazz) → one isolated collision shown
  in full → a cross-collision → a ranked pool of 8 angles → discards → the
  falsifier checks.

Each example carries its named references intact — the rule that separates a
real mechanism from a decorative analogy.
